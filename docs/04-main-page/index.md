# Главный экран содержит:

### Данные клиента:

- **Номер телефона** с помощью которого вошли в приложение;
- **ФИО** клиента на которого зарегистрирован номер телефона.

### Текущий баланс

### Тарифный план:

- **Наименование** тарифного плана
- **Остаток ресурсов** (гигабайты и минуты)
